The Discord Manifesto

By Michael Foster

A document detailing various ways to improve the runnings of the group Discord chat, timeline, reasons for the changes, etc.

Section 1: Rankings

Our current ranking system provides some customization of rankings– a 3 level system like our own runs well for the most part, but sometimes you need a few more roles so more people can have some kind of privilege. Because there are people in the Discord who deserve to have some control, but leveling them up to Mod rank is just too much. Also, if someone has done something that warrants punishment, it is extremely difficult to punish them because of the fact that @everyone rights are very difficult to revoke, even from specific people, and every right must be removed manually.

So what’s the solution? Here is the ranking system I suggest:

Rank Suggestions
Each rank has all the rights of all the level below it as well.

  1. @everyone - absolutely no permissions
  2. @Time Out - *a level with minimal permissions. negotiable, but I’m thinking the following:
    • read messages
    • join #general
  3. @normal - the average user. what @everyone is right now.
  4. @Mod1 - a very low level mod with some basic nominal rights.
    • manage emojis
    • send TTS
    • mention @everyone
  5. @Mod2 - slightly higher level moderator with some voice management rights.
    • server mute users
    • server deafen users
    • move users to different voice channels
  6. @Manager1 - control over chat and a few other permissions.
    • pin messages
    • delete messages
    • manage nicknames
    • manage webhooks (don’t think this will mean much to most people but it will to a few)
    • kick users
  7. @Manager2 - starts adding control over server settings.
    • add/edit/delete chatrooms
    • add/edit/delete voice channels
    • ban users
  8. @Admin - every control except editing their own role
  9. @owner - not a real role, but Nathan has control over actually everything
