Setup for LAN recording

For users without Audacity and OBS:
1. Click the button below to download the ZIP file.
2. Extract the files into a folder.
3. Run "Start Setup.bat".
4. Follow the instructions written there.

For users without Audacity and OBS:
1. Click the button below to download the ZIP file.
2. Extract the files into a folder.
3. In the "installers" folder, run the installers for any programs you haven't downloaded.
4. In the main directory, run "Load Settings.bat".
